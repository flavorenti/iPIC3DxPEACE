Set the boundary condition on 3D box last boundary cell using field communicator.

Function defining the rule to give value to last cell in your domain (e.g. V[0][j][k] in the case of Xleft). This rule can be (i) asymmetric V[0]=-V[1] using flag=0, (ii) null V[0]=0 using flag=1, (iii) symmetric V[0]=V[1] using flag=2. The value of flag must be specified in the input file.

Parameters
[in]nxdimension of the box in X from inp file
[in]nydimension of the box in Y
[in]nzdimension of the box in Z
[out]vector3-pointer to be updates at boundaries
[in]bcFaceXrightflag for BC anti-sunward (outflow side) from inp file
[in]bcFaceXleftflag for BC sunward (inflow side)
[in]bcFaceYrightflag for BC (dusk side)
[in]bcFaceYleftflag for BC (dawn side)
[in]bcFaceZrightflag for BC (north side)
[in]bcFaceZleftflag for BC (south side)
[in]vctbuilt-in type defining the 3D MPI cartesian topology
